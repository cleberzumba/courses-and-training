**Institution:** TrendyTech (India)<br>
**Instructor:** Sumit Mittal<br>

## Overview

Intensive industry program in Cloud Data Engineering, structured in 3 milestones, each followed by a project and interview preparation module.

## Course Content

### Milestone 1 (Weeks 1–14) — Distributed Processing with PySpark

1. **Understanding Big Data — The Big Picture**
2. **Distributed Storage — Data Lake Fundamentals**
3. **Distributed Processing Fundamentals — Apache Spark**
4. **Apache Spark Core APIs**
5. **Spark Higher-Level APIs — DataFrames & Spark SQL**
6. **Spark DataFrame Transformations**
7. **Apache Spark Caching In-Depth**
8. **Apache Spark Architecture**
9. **Apache Spark Internals**
10. **Apache Spark Optimizations — Part 1**
11. **Apache Spark Optimizations — Part 2**
12. **Apache Spark Project — Part 1**
13. **Apache Spark Project — Part 2** *(with unit testing)*
14. **Git, GitHub & CI/CD**
15. **Apache Hive**

### Milestone 2 (Weeks 15–23) — Azure Cloud Data Engineering

16. **Azure Cloud Fundamentals**
17. **Azure Storage & Databricks Fundamentals**
18. **Azure Databricks — Delta Lake**

20. **Azure Databricks — Unity Catalog**
21. **Azure Data Factory — Part 1**

### Milestone 3 (Weeks 24–34) — Streaming, Autoloader & AWS
