# Oracle GoldenGate Módulo I

**Institution:** GoldenGate BR (Gilson Martins)
**Certificate:** [certificate.pdf](./certificate.pdf)
**Workload:** 24 hours

## Course Content

### Session 1
1. **Introduction to Oracle GoldenGate** — History, what OGG is, implementation approaches, and common use cases.
2. **Architecture** — OGG architecture, supported databases and operating systems, product line, process groups, parameter files, GGSCI, and Integrated vs. Classic Extract modes.
3. **Installation** — Prerequisites, downloading and installing binaries, database configuration, and environment variables.
4. **Environment Preparation** — Enabling OGG in the database, configuring supplemental logging, and starting the Manager process.
5. **OGG Configuration** — Manager process, process definitions, parameter definitions, table definitions, and DEFGEN.

### Session 2
6. **Extract Configuration** — Classic and Integrated Extract modes, Data Pump configuration, and Extract parameter files.
7. **Initial Load Configuration** — Prerequisites, limitations, and methods for performing an initial data load.
8. **Replicat Configuration** — Classic and Integrated Replicat modes, and Replicat setup.
9. **Trail File Management** — Trail file content, format, header, and data structure.
10. **OGG Parameters** — Best practices and parameter configuration for Manager, Extract, Pump, and Replicat processes, including macros.

### Session 3
11. **DML and DDL Replication** — Performing DML and DDL replication, and monitoring transactions with GGSCI.
12. **Administration Commands** — Task automation with OBEY files, GGSCI commands, and configuration standardization.
13. **Monitoring Commands** — GGSCI commands, report and discard files, OGG logs, syslog, lag/checkpoint verification.
14. **Additional Information (Advanced Topics)** — Integrated Replicat, converting Classic to Integrated mode, Coordinated Replicat, Zero Downtime migration, encryption, and Credential Store/Wallet.
15. **Final Tips and Graphical Tools** — Overview of OGG Studio, OGG Monitor, OGG Plug-in for EMCC, OGG Veridata, and final recommendations.
