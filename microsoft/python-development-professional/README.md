**Institution:** Microsoft<br>

## Course Content

1. **Introduction to Python** — Python syntax and core language features; setting up the development environment; the difference between scripts and full programs; writing and running first Python programs.
2. **Python Basics** — Data types (integers, floats, strings), operators, variables, and control flow statements for building dynamic programs.
3. **Functions and Modules** — Writing reusable, organized code using functions; creating and importing modules; distinguishing built-in modules from external libraries.
4. **Python Data Structures** — Lists, tuples, dictionaries, and sets; storing and manipulating data effectively; implementing basic algorithms such as sorting and searching.
5. **Error Handling and Debugging** — Identifying and fixing common Python exceptions; using exception handling to manage errors gracefully; debugging techniques and tools.
6. **Testing and Version Control Basics** — Writing unit tests with pytest; fundamentals of Git for version control and collaboration.
