# Oracle Backup and Recovery

**Institution:** Nerv Informática (Ricardo Portilho)
**Certificate:** [certificate.pdf](./certificate.pdf)

## Objectives

- Get started with RMAN: configuring ARCHIVELOG mode, the CATALOG, creating scripts, and maintaining RMAN in good working order.
- Learn and use all RMAN configuration and maintenance options.
- Learn the different retention and backup administration strategies.
- Validate a backup and the database itself using RMAN.
- Check which backups are available in RMAN and how to add previously executed backups to the CATALOG.
- Perform every type of backup RMAN supports: cold, hot, compressed, full, partial, differential, incremental, etc.
- Learn the main recovery scenarios through simulated data-loss situations.
- Simulate and recover from all possible recovery scenarios: loss of DATABASE, TABLESPACE, DATAFILEs, TEMPFILEs, UNDO, Rollback Segments, CONTROLFILEs, ONLINE REDO LOGs, ARCHIVED REDO LOGs, PFILE, and SPFILE.
- Learn and practice RESETLOGS and Point-in-Time Recovery concepts.
- Simulate disaster scenarios, such as losing both the database and its backup, or losing archive logs, and recover from them.
- Use advanced RMAN 11g features such as CLONE, TSPITR, Data Recovery Advisor, and more.
