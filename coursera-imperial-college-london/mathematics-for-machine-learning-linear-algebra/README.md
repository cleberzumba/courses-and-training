# Mathematics for Machine Learning: Linear Algebra

**Institution:** Imperial College London (via Coursera)<br>
**Instructors:** David Dye, Samuel J. Cooper, A. Freddie Page<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Completed:** September 2025<br>

## Overview

Course covering the foundations of linear algebra applied to Machine Learning, including vector and matrix operations, transformations, orthogonality, and eigenanalysis, with hands-on Python/NumPy programming assignments.

## Programming Assignments

- **Identifying Special Matrices** — Implementing a function to determine whether a 4×4 matrix is singular, using row reduction to echelon form.
- **Gram-Schmidt Process** — Implementing the Gram-Schmidt procedure to build an orthonormal basis from a set of vectors, and determining the dimension of the space they span.
- **Reflecting Bear** — Applying matrix transformations (reflections) to 2D data using linear algebra operations.
