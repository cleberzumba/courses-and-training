# Microsoft Power BI for Business Intelligence and Data Science

**Institution:** Data Science Academy<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** 72 hours

## Course Content

1. **Introduction** — Course overview and foundational concepts.
2. **First Steps with Microsoft Power BI** — Introduction to the Power BI interface and basic functionality.
3. **Modeling, Relationships, and DAX** — Data modeling, table relationships, and DAX formulas.
4. **Power BI for Marketing Data Analysis** — Applying Power BI to analyze marketing data.
5. **Power BI for Sales Data Analysis** — Applying Power BI to analyze sales data.
6. **Power BI for HR Data Analysis** — Applying Power BI to analyze human resources data.
7. **Power BI for Logistics Data Analysis** — Applying Power BI to analyze logistics data.
8. **Power BI for Financial Data Analysis** — Applying Power BI to analyze financial data.
9. **Power BI for Accounting Data Analysis** — Applying Power BI to analyze accounting data.
10. **Power BI for Stock Market Data Analysis** — Applying Power BI to analyze stock market data.
11. **Fundamental Statistics for Data Science** — Core statistical concepts applied to data science.
12. **Data Cleaning and Manipulation with Power BI** — Techniques for cleaning and manipulating data within Power BI.
13. **Data Manipulation with Power Query M Language** — Using the M language in Power Query for data manipulation.
14. **Power BI and Databases** — Connecting and working with databases in Power BI.
15. **SQL Analytics** — Applying SQL for analytics within the Power BI workflow.
16. **Machine Learning and Power BI for Customer Segmentation** — Applying Machine Learning with Power BI for customer segmentation.
17. **Machine Learning and Power BI for Anomaly Detection** — Applying Machine Learning with Power BI for anomaly detection.
18. **Artificial Intelligence and Time Series Analysis with Power BI** — Applying AI and time series analysis techniques within Power BI.
19. **Case Study: Digital Transformation with Microsoft Power Platform** — Applied case study on digital transformation using the Microsoft Power Platform.
20. **Evaluation and Completion Certificate** — Final course evaluation and certificate of completion.
