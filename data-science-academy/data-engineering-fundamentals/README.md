# Data Engineering Fundamentals

**Institution:** Data Science Academy<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** 24 hours

## Course Content

1. **Data Pipelines and the Data Engineering Process** — Overview of data pipelines and the overall data engineering process.
2. **Data Pipeline Architectures** — Common architectural patterns for designing data pipelines.
3. **Distributed Storage and Processing** — Concepts of distributed data storage and processing.
4. **Data Warehouse, Data Lake, and Data Lakehouse** — Overview of data warehouse, data lake, and lakehouse architectures.
5. **Introduction to Data Modeling** — Core concepts of data modeling.
6. **Data Quality, Data Lineage, and Data Observability** — Concepts and practices for ensuring data quality, tracking lineage, and observability.
7. **DevOps for Data Engineers** — DevOps practices applied to data engineering workflows.
8. **Data Engineering with Python** — Applying Python to data engineering tasks.
