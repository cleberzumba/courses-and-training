# Big Data Real-Time Analytics with Python and Spark

**Institution:** Data Science Academy<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** 72 hours

## Course Content

1. **Introduction** — Course overview and foundational concepts.
2. **Data Manipulation with Python — NumPy** — Data manipulation techniques using NumPy.
3. **Data Manipulation with Python — Pandas** — Data manipulation techniques using Pandas.
4. **Exploratory Data Analysis** — Techniques for exploring and understanding datasets.
5. **Feature Engineering** — Techniques for creating and selecting features for analysis and modeling.
6. **Data Preprocessing** — Data cleaning and preparation techniques prior to analysis or modeling.
7. **Statistical Data Analysis — Part 1** — Statistical analysis techniques applied to data (part 1).
8. **Statistical Data Analysis — Part 2** — Statistical analysis techniques applied to data (part 2).
9. **Machine Learning with Python — Part 1** — Machine Learning fundamentals and implementation in Python (part 1).
10. **Machine Learning with Python — Part 2** — Machine Learning fundamentals and implementation in Python (part 2).
11. **Data Processing with Apache Spark** — Large-scale data processing using Apache Spark.
12. **Apache Spark SQL — Part 1** — Querying and processing data using Spark SQL (part 1).
13. **Apache Spark SQL — Part 2** — Querying and processing data using Spark SQL (part 2).
14. **Apache Spark Machine Learning** — Building Machine Learning models using Spark MLlib.
15. **Real-Time Analytics with Apache Spark Streaming — Part 1** — Real-time data processing using Spark Streaming (part 1).
16. **Real-Time Analytics with Apache Spark Streaming — Part 2** — Real-time data processing using Spark Streaming (part 2).
17. **Bonus: SQL Language with PySpark on Google Colab** *(Bonus)* — Applied bonus content using SQL with PySpark in Google Colab.
18. **Projects with Feedback** — Applied projects with instructor feedback.
