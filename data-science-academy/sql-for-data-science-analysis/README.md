1. **Introduction** — Course overview and foundational concepts.

4. **SQL Fundamentals — Part 1** — Core SQL fundamentals (part 1).

8. **SQL Queries with Table Joins — Part 2** — Writing queries using table joins (part 2).
9. **Working with Aggregation for Data Analysis — Part 1** — Using aggregation functions for data analysis (part 1).
10. **Working with Aggregation for Data Analysis — Part 2** — Using aggregation functions for data analysis (part 2).
11. **Window Functions and Subqueries — Part 1** — Using window functions and subqueries (part 1).
12. **Window Functions and Subqueries — Part 2** — Using window functions and subqueries (part 2).
13. **Exploratory Data Analysis with SQL** — Performing exploratory data analysis using SQL.
14. **Data Cleaning and Transformation Tasks with SQL** — Data cleaning and transformation techniques using SQL.
15. **Data Cleaning and Processing with SQL — Part 1** — Data cleaning and processing practice (part 1).
16. **Data Cleaning and Processing with SQL — Part 2** — Data cleaning and processing practice (part 2).
17. **Data Analysis with SQL and the Selection Process for Data Analysts** — Applying SQL data analysis in the context of the data analyst selection process.
18. **Database Programming with SQL — Part 1** — Database programming techniques using SQL (part 1).
19. **Database Programming with SQL — Part 2** — Database programming techniques using SQL (part 2).
20. **SQL Query Optimization — Part 1** — Techniques for optimizing SQL queries (part 1).
21. **SQL Query Optimization — Part 2** — Techniques for optimizing SQL queries (part 2).
22. **Special Project: Data Analysis with Artificial Intelligence — Part 1** — Applied special project combining data analysis with AI (part 1).
23. **Special Project: Data Analysis with Artificial Intelligence — Part 2** — Applied special project combining data analysis with AI (part 2).
