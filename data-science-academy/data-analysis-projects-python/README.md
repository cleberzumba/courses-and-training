# Data Analysis Projects with Python

**Institution:** Data Science Academy<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** 82 hours

## Course Content

1. **Introduction** — Course overview and foundational concepts.
2. **Setting Up the Work Environment** — Preparing the development environment for the course.
3. **Extracting and Analyzing Data from Databases** — Techniques for extracting and analyzing data stored in databases.
4. **Data Processing Techniques for Analysis** — Core techniques for processing data prior to analysis.
5. **Data Cleaning and Missing Values Handling — Part 1** — Techniques for cleaning data and handling missing values (part 1).
6. **Data Cleaning and Missing Values Handling — Part 2** — Techniques for cleaning data and handling missing values (part 2).
7. **Exploratory Analysis and Statistical Chart Interpretation — Part 1** — Exploratory data analysis and interpretation of statistical charts (part 1).
8. **Exploratory Analysis and Statistical Chart Interpretation — Part 2** — Exploratory data analysis and interpretation of statistical charts (part 2).
9. **Encoding Techniques for Predictive Modeling** — Encoding techniques to prepare data for predictive modeling.
10. **Market Basket Analysis Association Rules — Part 1** — Association rule techniques for market basket analysis (part 1).
11. **Market Basket Analysis Association Rules — Part 2** — Association rule techniques for market basket analysis (part 2).
12. **Feature Engineering Techniques — Part 1** — Techniques for engineering features for analysis and modeling (part 1).
13. **Feature Engineering Techniques — Part 2** — Techniques for engineering features for analysis and modeling (part 2).
14. **Knowledge Checkpoint** — Review checkpoint consolidating concepts covered so far.
15. **Data Visualization Techniques** — Techniques for visualizing data and analysis results.
16. **Statistical Analysis Reporting with Python** — Building statistical analysis reports using Python.
17. **Advanced Data Analysis with ChatGPT** — Applying ChatGPT to support advanced data analysis workflows.
