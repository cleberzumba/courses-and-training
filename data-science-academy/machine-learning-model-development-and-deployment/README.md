# Machine Learning Model Development and Deployment

**Institution:** Data Science Academy<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** 100 hours

## Course Content

1. **Introduction** — Course overview and foundational concepts.
2. **Environment Setup** — Preparing the development environment for the course.
3. **The Machine Learning Model Building Process** — Overview of the end-to-end process for building Machine Learning models.
4. **Designing the Training and Testing Process — Part 1** — Designing the model training and testing workflow (part 1).
5. **Designing the Training and Testing Process — Part 2** — Designing the model training and testing workflow (part 2).
6. **Designing the Model Deployment Process — Part 1** — Designing the model deployment workflow (part 1).
7. **Designing the Model Deployment Process — Part 2** — Designing the model deployment workflow (part 2).
8. **Deploying the Model as an API via Docker** — Deploying a Machine Learning model as an API using Docker.
9. **Knowledge Checkpoint** — Review checkpoint consolidating concepts covered so far.
10. **Feature Engineering and Feature Store** — Feature engineering techniques and the use of a Feature Store.
11. **Model Retraining and Versioning** — Strategies for retraining and versioning Machine Learning models.
12. **Model and Data Drift** — Detecting and handling model and data drift in production.
