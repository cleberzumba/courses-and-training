# Data Warehouse Design and Implementation

**Institution:** Data Science Academy
**Part of:** Data Engineer Training (Formação Engenheiro de Dados)
**Workload:** — 64 hours

## Course Content

1. **Introduction** — Course overview and foundational concepts.
2. **Business Intelligence and Data Modeling** — Introduction to Business Intelligence concepts and their relationship with data modeling.
3. **Entity-Relationship Modeling** — Entity-relationship modeling techniques for database design.
4. **Dimensional Modeling** — Dimensional modeling concepts for data warehousing (star schema, snowflake schema, facts and dimensions).
5. **Oracle Database — Part 1** — Introduction to Oracle Database fundamentals.
6. **Oracle Database — Part 2** — Continued Oracle Database concepts and practice.
7. **ETL: Data Extraction, Transformation, and Loading** — ETL processes for extracting, transforming, and loading data.
8. **Data Warehouse Implementation Project — Part 1** — Hands-on project implementing a data warehouse (part 1).
9. **Data Warehouse Implementation Project — Part 2** — Hands-on project implementing a data warehouse (part 2).
10. **Implementing a Data Warehouse in the Cloud — Part 1** — Cloud-based data warehouse implementation (part 1).
11. **Implementing a Data Warehouse in the Cloud — Part 2** — Cloud-based data warehouse implementation (part 2).
