# Analytics, Visualization, and Decision-Making with Big Data

**Institution:** Data Science Academy<br>
**Part of:** Data Engineer Training (Formação Engenheiro de Dados)<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** — 54 hours

## Course Content

1. **Introduction** — Course overview and foundational concepts.
2. **Analytics with Scala — Part 1** — Data analytics techniques using the Scala language (part 1).
3. **Analytics with Scala — Part 2** — Data analytics techniques using the Scala language (part 2).
4. **Analytics with Java — Part 1** — Data analytics techniques using the Java language (part 1).
5. **Analytics with Java — Part 2** — Data analytics techniques using the Java language (part 2).
6. **Data Engineering with Apache Cassandra** — Data engineering practices using Apache Cassandra.
7. **Data Engineering with MongoDB** — Data engineering practices using MongoDB.
8. **Data Pipelines with Apache Airflow** — Building and orchestrating data pipelines using Apache Airflow.
9. **Azure Data Factory** — Building and managing data pipelines using Azure Data Factory.
10. **Data Processing for Decision-Making with Databricks** — Processing data to support decision-making using Databricks.
11. **Using Power BI to Extract Reports from Apache Hadoop** — Extracting and visualizing reports from Apache Hadoop data using Power BI.
