# Machine Learning and AI in Distributed Environments

**Institution:** Data Science Academy<br>
**Part of:** Data Engineer Training (Formação Engenheiro de Dados)<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** — 64 hours

## Course Content

1. **Deep Learning and Artificial Intelligence** — Fundamentals of Deep Learning and Artificial Intelligence concepts.
2. **ElasticSearch in Distributed Environments** — Using ElasticSearch for search and analytics in distributed environments.
3. **Machine Learning and AI with Docker** — Deploying Machine Learning and AI workloads using Docker containers.
4. **Machine Learning at Large Scale** — Techniques and architectures for training and deploying Machine Learning models at scale.
5. **Microservices** — Microservices architecture concepts applied to distributed data and ML systems.
