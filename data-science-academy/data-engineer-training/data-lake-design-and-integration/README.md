# Data Lake Design and Integration

**Institution:** Data Science Academy<br>
**Part of:** Data Engineer Training (Formação Engenheiro de Dados)<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** — 72 hours

## Course Content

1. **Introduction** — Course overview and foundational concepts.
2. **What Are Data Lakes** — Core concepts of data lakes and their role in modern data architectures.
3. **Data Lake Design — Part 1** — Data lake design principles and architecture (part 1).
4. **Data Lake Design — Part 2** — Data lake design principles and architecture (part 2).
5. **Data Lake Batch Data Acquisition** — Acquiring data into the data lake using batch processing.
6. **Data Lake Streaming Data Acquisition** — Acquiring data into the data lake using streaming processing.
7. **Data Lake Messaging Layer** — Messaging layer architecture and technologies within the data lake.
8. **Data Lake Data Storage** — Storage strategies and technologies for data lakes.
9. **Data Lake Deploy and Rollout** — Deployment and rollout process for a data lake solution.
10. **Adding Layers to the Data Lake** — Expanding the data lake with additional architectural layers.
11. **Bonus: Building a Serverless Data Lake** *(Bonus)* — Applied bonus content on building a serverless data lake architecture.
