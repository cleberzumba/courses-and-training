# Data Pipeline Orchestration with Apache Airflow

**Institution:** Data Science Academy<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** 84 hours

## Course Content

1. **Introduction** — Course overview and foundational concepts.
2. **Apache Airflow Fundamentals** — Core concepts and fundamentals of Apache Airflow.
3. **Setting Up the Work Environment with Airflow** — Preparing the development environment for working with Airflow.
4. **DAG Architecture and Development — Part 1** — Designing and developing DAGs (part 1).
5. **DAG Architecture and Development — Part 2** — Designing and developing DAGs (part 2).
6. **Extracting Data via API** — Techniques for extracting data through APIs within Airflow pipelines.
7. **Integration Between Apache Airflow and Apache Spark — Part 1** — Integrating Apache Airflow with Apache Spark (part 1).
8. **Integration Between Apache Airflow and Apache Spark — Part 2** — Integrating Apache Airflow with Apache Spark (part 2).
9. **Knowledge Checkpoint** — Review checkpoint consolidating concepts covered so far.
10. **Airflow and Cloud Computing** — Using Airflow in cloud computing environments.
11. **Special Module: AI-Based Personal Assistants** — Building AI-based personal assistants as a special module topic.
12. **Working with Hooks for Database Access** — Using Airflow Hooks to access databases.
13. **Working with Variables** — Using Airflow Variables within pipelines.
14. **Apache Airflow Deploy with Generative AI** — Deploying Apache Airflow workflows leveraging Generative AI.
