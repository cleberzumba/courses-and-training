# Data Engineering with Hadoop and Spark

**Institution:** Data Science Academy<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** 44 hours

## Course Content

1. **Introduction** — Course overview and foundational concepts.
2. **Installing the Hadoop Ecosystem — Part 1** — Installing the Hadoop ecosystem (part 1).
3. **Installing the Hadoop Ecosystem — Part 2** — Installing the Hadoop ecosystem (part 2).
4. **Planning and Configuring a Hadoop Cluster** — Planning and configuring a Hadoop cluster.
5. **Using MapReduce on Large Data Volumes** — Applying MapReduce for processing large volumes of data.
6. **Data Storage with HBase and Hive — Part 1** — Data storage techniques using HBase and Hive (part 1).
7. **Data Storage with HBase and Hive — Part 2** — Data storage techniques using HBase and Hive (part 2).
8. **ETL Connectivity with the Hadoop System** — Connecting ETL processes to the Hadoop ecosystem.
9. **Hadoop Administration and Maintenance** — Administration and maintenance practices for Hadoop.
10. **Hadoop Machine Learning with Apache Mahout** — Applying Machine Learning on Hadoop using Apache Mahout.
11. **Apache Hadoop and Apache Spark** — Integrating Apache Hadoop and Apache Spark.
12. **Processing and Analyzing Billions of Records with Presto, Hive, AWS EMR, and Elastic MapReduce** — Processing and analyzing billions of records using Presto, Hive, AWS EMR, and Elastic MapReduce.

### Mini-Projects

- **Mini-Project 1: Importing Data from Oracle to HDFS** — Applied mini-project importing data from Oracle into HDFS.
- **Mini-Project 2: Predicting Cardiac Disease Cases** — Applied mini-project predicting cardiac disease cases.
- **Mini-Project 3: Designing a MapReduce Job with Spark** — Applied mini-project designing a MapReduce job using Spark.
