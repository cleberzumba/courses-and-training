# Fundamentals of Probability and Statistics for Data Science

**Institution:** USP (Universidade de São Paulo)<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** 30 hours

## Course Content

1. **Probability** — Fundamentals of probability theory.
2. **Conditional Probability and Bayes' Theorem** — Conditional probability concepts and Bayes' Theorem.
3. **Random Variables** — Introduction to random variables.
4. **Expectation, Variance, and Cumulative Distribution** — Expected value, variance, and cumulative distribution functions.
5. **Discrete Models** — Discrete probability distribution models.
6. **Continuous Models** — Continuous probability distribution models.
7. **Multidimensional Random Variables** — Random variables in multiple dimensions.
8. **Expectation and Covariance** — Expected value and covariance concepts.
9. **Monte Carlo Simulation** — Simulation techniques using the Monte Carlo method.
10. **Exploratory Data Analysis** — Techniques for exploring and understanding datasets.
11. **Personal Estimation** — Estimation techniques and personal application exercises.
12. **Limit Theorems** — Key limit theorems in probability and statistics.
13. **Confidence Intervals** — Constructing and interpreting confidence intervals.
14. **Hypothesis Testing** — Fundamentals of statistical hypothesis testing.
15. **Chi-Square Test, Paired Tests, and Kolmogorov-Smirnov Test** — Chi-square testing, paired tests, and the Kolmogorov-Smirnov test.
16. **Regression Model** — Fundamentals of regression modeling.
17. **Supervised Learning** — Introduction to supervised learning techniques.
18. **Model Selection and Regularization** — Techniques for model selection and regularization.
19. **Bayesian Inference** — Fundamentals of Bayesian inference.
20. **KNN and Logistic Regression** — K-Nearest Neighbors and logistic regression models.
21. **Project** — Applied final project consolidating course concepts.
