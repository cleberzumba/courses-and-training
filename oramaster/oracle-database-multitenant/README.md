# Oracle Database Multitenant

**Institution:** Oramaster<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** — 6 hours<br>

## Course Content

1. **Introduction to Multitenant Architecture** — Overview, benefits, and licensing of Oracle's Multitenant Architecture; Container Database (CDB) and Pluggable Database (PDB) concepts.
2. **Oracle Database Structure in the Multitenant Architecture** — Structural differences between non-CDB and CDB/PDB architectures.
3. **Creating Pluggable Databases** — Methods and procedures for creating pluggable databases (PDBs).
4. **Managing Pluggable Databases** — Administration tasks for managing PDBs (open/close, plug/unplug, cloning).
5. **Users and Security in the Multitenant Architecture** — Managing common and local users, roles, and privileges in a multitenant environment.
6. **Backup and Recovery in the Multitenant Architecture** — Backup and recovery strategies specific to CDB/PDB environments using RMAN.
