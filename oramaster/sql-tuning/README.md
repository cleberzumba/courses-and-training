# Oracle SQL Tuning

**Institution:** Oramaster
**Certificate:** [certificate.pdf](./certificate.pdf)
**Workload:** 24 hours

## Course Content

### Session 1 (8 hours)
1. **Introduction** — SQL Tuning fundamentals, objectives, and where performance problems typically occur.
2. **Database Design** — Impact of data modeling and database design on SQL performance.
3. **Understanding the Optimizer and Execution Plans** — How the Oracle Optimizer works and how to read execution plans.
4. **Object Statistics** — Role of optimizer statistics in query performance and execution plan selection.

### Session 2 (8 hours)
5. **Optimizing DML** — Techniques for tuning INSERT, UPDATE, and DELETE statements.
6. **Optimizing Queries** — Techniques for tuning SELECT statements and query performance.

### Session 3 (4 hours)
7. **Influencing the Optimizer** — Using hints and other techniques to influence optimizer decisions.
8. **Troubleshooting** — Diagnosing and resolving common SQL performance problems.

### Session 4 (4 hours)
9. **Final Challenge** — Applied exercise consolidating all SQL tuning concepts covered.
10. **Closing** — Course wrap-up and final recommendations.
