# Oracle Database Performance Tuning

**Institution:** Oramaster<br>
**Certificate:** [certificate.pdf](./certificate.pdf)<br>
**Workload:** — 24 hours<br>

## Course Content

1. **Investigating Problems** — Techniques and approaches for investigating and diagnosing database performance problems.
2. **Wait Events** — Analyzing wait events to identify bottlenecks and root causes of performance issues.
3. **AWR (Automatic Workload Repository)** — Using AWR reports and snapshots to monitor and diagnose database performance over time.
4. **Instance Tuning** — Tuning techniques focused on database instance-level performance (memory, processes).
5. **Database Tuning** — Tuning techniques focused on database-level performance (SQL, storage, structures).
