# Data Engineering with Microsoft Azure

**Institution:** Udacity
**Certificate:** [certificate.pdf](./certificate.pdf)
**Workload:** 55 hours

## Course Content

### 1. Data Modeling (9 hours)

- **Introduction to Data Modeling** — Core concepts of data modeling, when and why to model data for different use cases.
- **Relational Data Models** — Relational database design, normalization, and modeling for OLTP/OLAP systems.
- **NoSQL Data Models** — Modeling for NoSQL databases, use cases where NoSQL is preferable over relational models.

### 2. Cloud Data Warehouses with Azure (14 hours)

- **Introduction to Cloud Data Warehouses with Azure** — Overview of cloud data warehousing concepts and Azure's role in modern data architectures.
- **Introduction to Data Warehouses** — Core data warehouse concepts, OLAP vs. OLTP, and warehouse design fundamentals.
- **ELT and Data Warehouse Technology in the Cloud** — ELT pipeline design and technologies for moving data into cloud data warehouses.
- **Azure Data Warehouse Technologies** — Overview of Azure-specific services for data warehousing (e.g., Synapse Analytics).
- **Implementing Data Warehouses in Azure** — Hands-on implementation of a data warehouse solution using Azure services.
- **Building an Azure Data Warehouse for Bike Share Data Analytics** *(Project)* — Applied project building a data warehouse for bike share analytics on Azure.

### 3. Data Lakes and Lakehouses with Spark and Azure Databricks (14 hours)

- **Course Introduction** — Overview of the module's goals and structure.
- **Big Data Ecosystem, Data Lakes, and Spark** — Introduction to the big data ecosystem, data lake concepts, and Apache Spark fundamentals.
- **Data Wrangling with Spark** — Data cleaning and transformation techniques using Spark.
- **Spark Debugging and Optimization** — Techniques for debugging and optimizing Spark jobs for performance.
- **Azure Databricks** — Introduction to the Azure Databricks platform and its core features.
- **Data Lakes and Lakehouse with Azure Databricks** — Building data lakes and lakehouse architecture using Azure Databricks.
- **Building an Azure Data Lake for Bike Share Data Analytics** *(Project)* — Applied project building a data lake for bike share analytics on Azure.

### 4. Data Pipelines with Azure (18 hours)

- **Introduction to Data Pipelines** — Fundamentals of data pipeline design and orchestration.
- **Azure Data Pipeline Components** — Overview of Azure Data Factory components used to build pipelines.
- **Transforming Data in Azure Data Pipelines** — Data transformation techniques within Azure pipeline workflows.
- **Azure Pipeline Data Quality** — Ensuring data quality and validation within pipeline workflows.
- **Azure Data Pipelines in Production** — Deploying, monitoring, and managing pipelines in production environments.
- **Data Integration Pipelines for NYC Payroll Data Analytics** *(Project)* — Applied project building integration pipelines for NYC payroll data analytics on Azure.
