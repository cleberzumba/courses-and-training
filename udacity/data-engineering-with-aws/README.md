# Data Engineering with AWS

**Institution:** Udacity
**Certificate:** [certificate.pdf](./certificate.pdf)
**Workload:** 66 hours

## Course Content

### 1. Data Modeling (18 hours)

- **Introduction to Data Modeling** — Core concepts of data modeling, when and why to model data for different use cases.
- **Relational Data Models** — Relational database design, normalization, and modeling for OLTP/OLAP systems.
- **NoSQL Data Models** — Modeling for NoSQL databases, use cases where NoSQL is preferable over relational models.
- **Data Modeling with Apache Cassandra** — Designing Cassandra data models based on query patterns (query-first design).

### 2. Cloud Data Warehouses (13 hours)

- **Introduction to Cloud Data Warehouses with AWS** — Overview of cloud data warehousing concepts and AWS's role in modern data architectures.
- **Introduction to Data Warehouses** — Core data warehouse concepts, OLAP vs. OLTP, and warehouse design fundamentals.
- **ETL and Data Warehouse Technology in the Cloud** — ETL pipeline design and technologies for moving data into cloud data warehouses.
- **AWS Data Warehouse Technologies** — Overview of AWS-specific services for data warehousing (e.g., Redshift).
- **Implementing a Data Warehouse on AWS** — Hands-on implementation of a data warehouse solution using AWS services.
- **Data Warehouse Project** — Applied project consolidating data modeling and warehousing concepts on AWS.

### 3. Spark and Data Lakes (15 hours)

- **Introduction to Data Lakes and Lakehouses** — Core concepts of data lakes and the lakehouse architecture, and how they differ from traditional warehouses.
- **Building Data Lakes on AWS** — Designing and implementing data lakes using AWS storage and cataloging services.
- **Processing Data in the Lake with Spark** — Large-scale data processing on data lakes using Apache Spark.
- **Implementing Lakehouse Tables** — Building and managing lakehouse table formats for combined warehouse/lake capabilities.

### 4. Automate Data Pipelines (20 hours)

- **Introduction to Data Pipelines and Airflow** — Fundamentals of data pipeline orchestration using Apache Airflow.
- **Data Lineage and Orchestration** — Tracking data lineage and orchestrating dependent tasks across pipelines.
- **Orchestrating Warehouse Workflows with Amazon Redshift** — Automating and scheduling data warehouse workflows on Redshift using Airflow.
- **Orchestrating Lakehouse Workflows with AWS Glue and Athena** — Automating lakehouse pipelines using AWS Glue and Athena, orchestrated via Airflow.
