# Oracle Exadata Advanced Experience

**Institution:** Lore Data (Franky)

## Objectives

- Understand Exadata architecture and its components.
- Learn Exadata's main capabilities and key features.
- Be able to administer and configure Oracle Exadata.
- Be able to create, administer, and configure Oracle Database on Oracle Exadata.

## Course Content

- History of Exadata
- Brief overview of Exadata's composition
- Exadata physical and logical architecture
- New features of the X8M, X9M, X10M, and X11M Exadata models
- Key benefits of Exadata
- Exadata Exascale
- Exadata Cloud (ExaCC and ExaCS)
- PMem, XRMem, RDMA
- Exadata models
- Exadata deployment process
- Exadata installation
- Performance features on Exadata
- Smart Scan and Offloading, Storage Index, and Flash Cache
- Storage Server administration
- Database Server administration
- Database administration on Exadata
- Exadata patching process
- Exadata monitoring
